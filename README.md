# KeepTheStyle v1

A browser-based visual website builder for designing, editing, and exporting web interfaces without writing code.

## Run locally

Open `index.html` in a modern browser. No build step is required.

## Current capabilities

- Drag-and-drop element library
- Visual canvas with selection, movement, resizing, and inline text editing
- Searchable professional property inspector
- Multi-selection, copy/paste, undo/redo, locking, visibility, and layer ordering
- HTML, CSS, and JavaScript preview/export
- Local project persistence and `.ktstyle` project files
- Light and dark workspace themes

## Development

The application uses plain HTML, CSS, and JavaScript. Source files are organized under `css/` and `js/`.
